Article 1: Overview of the Service

This service provides ephemeral chat rooms accessed using randomly generated hash values (hereinafter referred to as 'Room Identifiers').

  • Messages are automatically deleted up to 15 minutes after being sent.
  • No user registration, ID, or password setup is required.
  • Room Identifiers are 30-character random values based on SHA-256, making access virtually impossible for those without prior knowledge.
Article 2: Handling of Data

This service handles the following data:

1. Message Data

Deleted messages and images cannot be recovered.

2. Connection Information (Circumstantial Evidence)

For operation and legal compliance, the following information may be temporarily stored:

  • IP address
  • Connection timestamp
  • Room Identifier (e.g., the value of ?room=xxx)
  • User agent information (browser details of the device used)

In principle, this information is automatically deleted after 30 days.
